Xcode中常用的快捷键操作

'窗体'
 * command + 0
 * command + option + 0
 * command + shift + Y
 command  +  1~8
 command + option + 1~2
 command + option + control + 1~4
 control + 1~6
'调试'
 command + R
 * command + .
 command + B
 command + shift + K
'编辑'
 command + [
 command + ]
 command + /
 command + option + / 注释
 * command + option + ← 或 →
'文件切换'
 command + ~
 * command + control + ← 或 → 或 ↑ 或 ↓
'文件操作'
 command + O
 command + S
 * command + W
 * command + Q

 'Other'
 * command + shift + 0             快速打开
 * command + shift + o
 * Command + Shift + J             展示当前你在工程导航器中打开的文件
 option + leftClick                辅助编辑器中打开文件
 command + shift + F               搜索导航器
